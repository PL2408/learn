ESC - mode for commands

i - mode for Insert (type text)

:w - write changes
:q - quit

:wq - write and quit

:q! - done't write (save), just quit wihtout savings

# comments in bash scripts

Щоб перемістити курсор, натисніть клавіші h,j,k,l так, як показано нижче:
             ^
             k Поради: Клавіша h знаходиться ліворуч і переміщується ліворуч.
       < h l > Клавіша l знаходиться праворуч і рухається праворуч.
             j Клавіша j схожа на стрілку "вниз".
             v
<Ctrl-f> – на сторінку (екран) вниз;
<Ctrl-b> - на сторінку (екран) верх;
<Ctrl-d> - на підлогу сторінки (екрана) вниз;
<Ctrl-u> - на підлогу сторінки (екрана) верх;
<Ctrl-y> - на рядок вниз, без зміни положення курсору;
<Ctrl-e> - на рядок верх, без зміни положення курсору;
yy - copy line
p - paste copied line
u - undo
. - repeat last command
dd - delete line
0 («нуль») - на початок поточного рядка;
^ - на початок поточного рядка (до першого непробільного символу);
$ - у кінець поточного рядка;
w - на слово праворуч;
b - на слово вліво;
W - до пробілу вправо;
B - до пробілу вліво;
} - абзац донизу;
{- абзац вгору;
gg - перейти на початок файлу;
G – перейти в кінець файлу;
<number>G – перейти на конкретний рядок <number>;
/<text><CR> - перейти до <text>;
?<text><CR> - те саме, але шукати назад;
n – повторити пошук;
N – повторити пошук назад;
[[ - на початок функції;
'' - до місця виконання команди [[
