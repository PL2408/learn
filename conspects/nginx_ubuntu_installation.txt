Крок 1: Встановлення NGINX
Встановимо NGINX:

--- ✄ -------------------------

sudo apt update
sudo apt install nginx

Перевіримо, чи запущено NGINX:
sudo systemctl status nginx

--- ✄ -------------------------

Крок 2: Налаштування порту 8080 для NGINX
Відкрийте файл конфігурації NGINX:

sudo nano /etc/nginx/sites-available/default
Додайте або змініть рядок для слухання порту 8080:

listen 8080 default_server;
Збережіть файл та вийдіть з редактора.

Перезавантажте NGINX, щоб застосувати зміни:

sudo systemctl restart nginx
Перевірте, чи слухає NGINX порт 8080:

sudo ss -lntp | grep 8080

--- ✄ -------------------------

Крок 3: Створення статичної HTML-сторінки
Створимо каталог для нашої статичної сторінки:

sudo mkdir -p /var/www/html
Створимо та редагуємо файл сторінки:

sudo nano /var/www/html/index.html
Додайте вміст, що відповідає заданому JSON:

{
  "name": "student",
  "os": "Ubuntu 20.04.3 LTS",
  "ip": "10.6.220.135",
  "cpu": "1",
  "memory": "1.6"
}

Збережіть файл та вийдіть з редактора.

--- ✄ -------------------------

Крок 4: Перевірка роботи
Відкрийте браузер або використовуйте curl для перевірки сторінки:

curl localhost:8080
